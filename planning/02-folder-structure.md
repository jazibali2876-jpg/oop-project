## 7. Where the OOP layers physically sit

| Concern | Folder | Notes |
|---|---|---|
| OOP graded layer | `src/domain/` | No ImGui, no SFML headers anywhere here |
| Per-frame UI | `src/ui/` | Owns no business state |
| Integration glue | `src/platform/` | The only folder that includes BOTH SFML 3 and ImGui headers |
| File I/O | `src/domain/persistence/` | The ONLY folder that opens `fstream`s |
| Cross-cutting | `src/util/` | Small, dependency-free helpers |

If a future Phase-2 PR adds an `#include <imgui.h>` inside `src/domain/`, that PR is
wrong by definition вЂ” the layer split has been violated.
